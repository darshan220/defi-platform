# Project Overview: Equivo DeFi Platform

**Equivo** (formerly Nexus) is a next-generation Decentralized Finance (DeFi) trading platform built with Next.js. The platform is designed to provide users with a seamless, premium experience for managing their digital assets, trading tokenized equities, and participating in advanced financial protocols.

## High-Level Architecture
The application is structured as a modern, responsive web application utilizing Next.js's App Router. The layout is anchored by a persistent, intuitive sidebar navigation and a dynamic "cube panel grid" layout system that organizes complex financial data into digestible, interactive modules.

## Key Domains
The platform is divided into several core functional areas:
- **Dashboard/Home**: A centralized hub for an overview of user assets and market conditions.
- **Deposit**: A specialized interface for depositing stablecoins, featuring interactive tabs for buying, staking, and unstaking, complete with live APY tracking.
- **Trade**: The primary exchange interface allowing users to trade tokenized private equity. It features detailed token rows and interactive trade application dialogs.
- **Borrow**: A lending/borrowing interface where users can leverage their equity as collateral to borrow assets.
- **Portfolio**: A comprehensive asset management view for users to track their holdings, performance, and transaction history.
- **Ecosystem, Card & Auctions**: Additional platform features expanding utility (e.g., linked debit cards, asset auctions).

## Design Philosophy
Equivo employs a sophisticated "glassmorphic" aesthetic combined with vibrant teal accents. A central pillar of the project is its **Theme-Aware Design System**. Every component is built using semantic CSS variables, ensuring perfect visual fidelity and contrast across both Light and Dark modes. The platform prioritizes responsiveness, ensuring a flawless experience from desktop to mobile.

## Development Status
The application features a robust UI foundation with standardized spacing, premium typography, and fluid transitions. Current efforts focus on refining interactive components (like the multi-wallet connection modal) and ensuring layout consistency across all DeFi modules.
